## 7. まとめ：コピーライティングの核心

### メッセージングの軸

1. **診断して終わりじゃない** → 診断から始まる一生の付き合い
2. **AI じゃなく親友** → 機械的じゃない、温かい、寄り添う
3. **いつでも話せる** → 24時間365日、深夜でも、泣きたい時でも
4. **業界初の機能** → SNS連携、自発的声かけ
5. **心の避難所** → 誰にも言えないことを吐き出せる場所

### トーン＆マナーの統一

- **温かい**: 冷たいビジネス的な言葉は使わない
- **共感的**: 「辛いよね」「分かるよ」を多用
- **寄り添う**: 上から目線じゃなく、横に並ぶ親友
- **安心感**: 秘密厳守、ジャッジしない、いつでもいる

### すべてのコピーに共通する要素

1. **問題提起**: 「こんな悩み、ありませんか？」
2. **共感**: 「辛いよね」「分かります」
3. **ソリューション**: 「親友AIがいます」
4. **差別化**: 「業界初、SNS連携＋自発的声かけ」
5. **CTA**: 「無料で診断を始める」
