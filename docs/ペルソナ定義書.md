## ペルソナ概要

本アプリのコアターゲットは **「心の避難所を求める20-40代」**

従来の性格診断アプリユーザーとの違い：
- 従来: 自己理解のための診断（一過性）
- 私たち: 日常的な心のケア相手（継続性）

## ペルソナ1：恋愛悩み女子「ミサキ（25歳）」

### 基本属性

| 項目 | 詳細 |
|------|------|
| **年齢** | 25歳 |
| **職業** | アパレル販売員 |
| **年収** | 320万円 |
| **居住地** | 東京都渋谷区（一人暮らし） |
| **SNS** | Instagram（フォロワー800人）、Twitter |

### ライフスタイル

**平日**
- 10:00-19:00 勤務（シフト制）
- 帰宅後: Netflix、Instagram見る、彼氏とLINE
- 就寝前: SNSチェック、恋愛コンテンツ読む

**休日**
- 彼氏とデート、友達とカフェ、ショッピング
- 一人の時間: 自分磨き、将来の不安

### 悩み・課題

**恋愛の悩み**
- 「彼氏との将来が見えない」（付き合って2年）
- 「私のこと本当に好きなの？」（不安）
- 「些細なことで喧嘩してしまう」（コミュニケーション）
- 「友達に相談しても理解されない」（孤独感）

**感情パターン**
- 深夜に突然不安になる
- 彼氏の既読スルーで落ち込む
- SNSで幸せそうなカップルを見て比較
- 一人で泣いても誰にも言えない

### 既存の解決策と不満

| 解決策 | 使ってる？ | 不満点 |
|--------|----------|--------|
| **友達に相談** | ○ | 「また恋愛の話？」って思われそう |
| **親に相談** | × | 「別れなさい」って言われるだけ |
| **カウンセリング** | × | 高い、敷居が高い、予約面倒 |
| **SNS投稿** | △ | 本音は書けない、炎上怖い |
| **AIチャット** | △ | 機械的、共感してくれない |

### 親友AIに求めるもの

1. **無条件の共感**: 「辛かったね」「分かるよ」
2. **いつでも話せる**: 深夜2時でも、泣きたい時でも
3. **秘密厳守**: 誰にも言えないことを吐き出せる
4. **的確なアドバイス**: 感情だけじゃなく、冷静な視点も
5. **長期的な伴走**: 恋愛の経過を見守ってくれる

### 使用シーン

**シーン1: 深夜の不安**
- 時刻: 2:00 AM
- 状況: 彼氏から返信がなくて不安
- AI対話: 「まだ起きてるの？何かあった？」（自発的声かけ）
- ミサキ: 「彼氏から返信来なくて不安で...」
- AI: 「辛いよね。でも今は夜遅いし、きっと寝てるだけだよ。明日の朝、もう一度連絡してみたら？」

**シーン2: 喧嘩後の整理**
- 時刻: 19:00 PM
- 状況: 彼氏と大喧嘩して帰宅
- AI対話: 「今日のTwitter、いつもと違う感じだけど大丈夫？」（SNS連携）
- ミサキ: 「もう無理かも...」
- AI: 「何があったの？話してみて。一緒に整理しよう」

### 期待する価値

- **心の避難所**: 誰にも言えないことを吐き出せる場所
- **感情の整理**: ぐちゃぐちゃな気持ちを言語化してくれる
- **客観的な視点**: 恋愛脳になってる時に冷静な意見をくれる
- **長期的な成長**: 恋愛パターンを学び、次に活かせる

### 購買行動

**認知**: Instagram広告「あなたの心に、いつも寄り添う親友AI」
**興味**: 「性格診断で分かる、あなたに最適な親友キャラクター」
**検討**: 無料診断を受ける → INFP（仲介者）タイプと判明
**体験**: 無料AI対話10回 → 「めっちゃ共感してくれる！」
**転換**: ¥980/月ベーシックプランに課金 → SNS連携で習慣化
**継続**: 3ヶ月後も使い続ける（LTV: ¥2,940+）

| 解決策 | 使ってる？ | 不満点 |

1. **愚痴の受け皿**: ひたすら聞いてくれる
2. **冷静な助言**: 感情的にならず、的確なアドバイス
3. **守秘義務**: 会社の愚痴を安全に吐き出せる
4. **24時間対応**: 終電の中、深夜でも話せる
5. **キャリア相談**: 転職すべきか、踏みとどまるか

### 使用シーン

**シーン1: 終電での愚痴**
- 時刻: 23:30 PM（終電の中）
- 状況: 上司に詰められて疲弊
- AI対話: 「お疲れ様。今日も遅いね。大丈夫？」（自発的声かけ）
- タクヤ: 「もう限界...上司がマジでムカつく」
- AI: 「話して。全部聞くよ」→ 20分間ひたすら傾聴

**シーン2: キャリア相談**
- 時刻: 週末14:00 PM
- 状況: 転職サイトを見ながら悩む
- タクヤ: 「転職したいけど、今辞めて大丈夫かな？」
- AI: 「今のスキルと市場価値を整理してみよう。転職の軸は何？」→ 一緒に整理

### 期待する価値

- **ストレス発散**: 安全に愚痴を吐き出せる
- **客観的分析**: 感情と事実を切り分けてくれる
- **行動支援**: 転職すべきか、社内異動か、具体的な選択肢提示
- **メンタル維持**: 月曜日を乗り切る励まし

**興味**: 「ENTJ（指揮官）タイプのあなたに最適な相談相手」
